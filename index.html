SIGNAL333 Publisher is an automated content publishing system for the SIGNAL333 ecosystem. It renders and publishes original educational short-form video content to TikTok via the Content Posting API — daily, without manual intervention.

How It Works
01

Script Selection

Randomly selects from a reservoir of original educational scripts across five content series. Duplicate prevention ensures no script repeats within a 30-day window.

02

Video Rendering

Renders a 1080×1920 MP4 short-form video with series-specific branding, typography, and animated sequences. Each video is unique to its series.

03

TikTok Publishing

Authenticates via OAuth 2.0 with PKCE, queries creator info, initialises upload, and publishes directly to the @signal333hq TikTok account via the Content Posting API.

04

Logging & Scheduling

Logs every published script with timestamp. Runs daily via Windows Task Scheduler. Fully automated — no manual steps after initial setup.
