Privacy Policy

SIGNAL333 Publisher | Last updated: June 2026

The application stores TikTok OAuth access tokens locally for authentication. No personal data is collected from the public. No user accounts are created. No third-party content is uploaded.

1. Overview

SIGNAL333 Publisher is an internal automated publishing tool operated by SIGNAL333 Ventures. This policy explains how the tool handles data in connection with its TikTok integration.

2. Data Handled

The only data processed is the TikTok OAuth access token for authorised TikTok accounts. This token is stored locally on the operator's machine and is used solely to authenticate publishing requests to the TikTok Content Posting API.

3. Data Not Collected

This tool does not collect names, email addresses, IP addresses, device information, behavioural data, or any other personally identifiable information from any individual. No public user accounts are created or supported. No third-party content is uploaded.

4. Third-Party Services

This tool connects to TikTok's Content Posting API. All interactions with TikTok are governed by TikTok's Privacy Policy.

5. Data Security

Access tokens are stored locally on the operator's machine and are never shared with third parties or stored in any external database.
